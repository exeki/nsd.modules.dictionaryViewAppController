Контроллер для встроенного приложения [dictionary-view-app](https://github.com/exeki/dictionary-view-app).
Требует модуль [web_api_components](https://github.com/exeki/nsd.modules.web_api_components) для работы.
Удалите импорт ApiPlaceholder перед запуском, иначе не скомпилируется, если вы не используете модуль плейсхолдер библиотеки [nsd.sdk.global_variables](https://github.com/exeki/nsd.sdk.global_variables).
